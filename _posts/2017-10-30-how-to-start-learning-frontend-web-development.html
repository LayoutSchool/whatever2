If you are just starting out learning web development, this guide is for you.

HTML5, CSS3, and JavaScript are the must-know technologies of the web. Therefore, even if your goal is to become, for example, a php developer, in order to succeed, you need to know quite a lot about these three frontend technologies.

In this guide, we'll look at how to learn enough as easy as possible, what pitfalls to avoid, and how to boost effectiveness.

Learning frontend web development can be thought of as achieving levels of competence. The following list describes these levels of competence as steps that should be achieved:

Level 1 front-end web developer

  1. HTML: Be able to list 20 most common HTML selectors, and know how to use them
  2. HTML: Understand HTML attributes like id, class, and data-*
  3. HTML: Know what to put in the head tag of an HTML document
  4. CSS: What is CSS, what it does, and 3 ways of adding it to the page
  5. CSS: Understand how CSS targeting works
  6. CSS: Describe how CSS ids differ from CSS classes
  7. CSS: Be able to understand and use 5 most basic selectors for CSS targeting
  8. CSS: What is responsive web design and how it is achieved in CSS
  9. JS: What is JS and how can we start using it in our web pages
  10. JS: How jQuery makes it easier to use JS in our web pages
  11. JS: Using jQuery, change the color of a button on click

Level 2 front-end web developer

  1. HTML: Understand the difference between HTML4 and HTML5
  2. HTML: What are semantic tags in HTML
  3. HTML: What is SVG
  4. CSS: Understand the CSS cascade
  5. CSS: Understand CSS specificity
  6. CSS: Know what pseudo-classes are and how to use them
  7. CSS: Understand the box model and be able to manipulate it
  8. CSS: Understand CSS positioning and be able to use it
  9. JS: Be able to preventDefault on button submit
  10. JS: Know JavaScript primitives: Strings, Numbers, Null, Undefined, Boolean, and Symbol
  11. JS: Using jQuery, make an element appear and disappear on click
